GeoEdit
Программа для редактирование данных ГТИ, зарегистрированных в программном комплексе GeoScape.

GeoEdit v2.0 - Полностью переписанная версия программы GeoEdit II. Изменен интерфейс под двухоконное редактирование. Левое окно - таблица, правое окно - диаграмма на 7 графиков.
Дальнейшая история версий:
2.1.х - Устранение багов
2.2.1 - Добавлены функции, не сразу перенесенные из GeoEdit II
2.2.2 - Добавлена функция Сканирования поля "Забой", общая оптимизация алгоритмов
2.2.3 - Добавлена функция Сглаживания графиков
2.3.0 - Добавлена функция "Растяжка/Сжатие тальблока"
2.3.1-2.3.3 - Мелкие усовершенствования интерфейса и функционала
2.3.4 - Усовершенствован алгоритм Рихтовки "просечек" и добавлена Рихтовка "Лестниц" (обычно на Уровнях). Убран баг с зависанием при Сглаживании снизу вверх.
2.3.5 - Убран баг с правкой температур, добавлена возможность инвертирования кривой при заполнении "по шаблону" (для этого в поле 'Коэффициент поставить значение "-1"' и затем функцией 'Увеличение/Уменьшение' вывести к нужным значениям)
2.3.6 - В функцию "Рихтовка" добавлена возможность задавать и сохранять ( кнопка <Сохранить пресет> ) значения по умолчанию для каждого параметра индивидуально.
         - Добавлена функция "Откат", позволяющая отменить последнюю операцию редактирования блока.
2.3.7 - В функцию перерасчета параметров добавлено автосуммирование газов Хроматографа. 
           Внимание: изменение значений в таблице врукопашную не влечет за собой перерасчет зависимых параметров! Чтобы перерасчет параметров произошел, нужно изменить параметры блоком (например, прибавить и вычесть одно и то же значение) или воспользоваться рисованием на диаграмме.
2.4.0 - Добавлена функция импорта данных Хроматографа в файл GeoScape. Если размер данных Хроматографа выходит за пределы текущего файла GeoScape, то программа оповещает об этом и нужно продолжить импорт в следующий файл.
2.4.1 - Добавлена возможность переключать параметр во время рисования путем клика на нужной линии средней кнопкой (колесом) мыши.
      - Улучшен алгоритм Рихтовки по просечкам.
2.4.2 - Вырезана лицензионная защита для исключения ложного срабатывания антивирусного софта.
2.5.1 - Добавлена функция проверки уставки при заполнении блока. Убраны мелкие баги. Оптимизация алгоритмов проверки правильности значений.
